# Email Signature Generator

A web-based tool for creating professional email signatures. Customize your signature with your name, title, company information, contact details, and social media or certification icons. Generate accessible HTML code that can be directly inserted into your email client, or download your signature as an image.

Built with Next.js and designed to be deployed as a [single page application](https://nextjs.org/docs/app/guides/single-page-applications).

View the application as deployed via github actions, [here](https://jaden-drury.github.io/email-signature-generator/)

## Features

- **HTML Export**: Generate clean, accessible HTML for use in email clients
- **Image Export**: Download your signature as PNG, JPEG, or SVG
- **Customization Options**:
  - Personal information (name, title, company, contact details)
  - Social media icons with custom links
  - Background colors and borders
  - Icon positioning and alignment
  - Custom font sizes
- **Live Preview**: See your changes in real-time as you design
